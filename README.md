# After Action Report Generator

Generate properly formatted After Action Reports following USMC TECOM format with MCCLL-compatible structure.

## Features

- **TECOM Format Compliant** - Follows HQBN Enclosure 12 AAR template exactly
- **Dynamic Topics** - Add/remove multiple IMPROVE and SUSTAIN topics
- **Naval Letter Format** - Proper header with SSIC, office code, and correspondence blocks
- **Multiple Export Options**:
  - Copy to clipboard (plain text)
  - Export as DOCX
  - Print-friendly preview
- **Offline Capable** - Works without internet connection (PWA)
- **Auto-Save** - Drafts saved automatically to browser storage
- **Theme Support** - Light, dark, and tactical (night) modes
- **Mobile Friendly** - Responsive design works on any device

## Usage

1. Fill in unit and correspondence information
2. Add IMPROVE topics (areas needing improvement)
3. Add SUSTAIN topics (areas that worked well)
4. Enter POC and signature information
5. Preview, copy, or export your AAR

## Format Reference

This tool follows the official format from:
- **TECOM HQBN Enclosure 12** - After Action Report Template
- **MCO 3504.1** - Marine Corps Lessons Learned Program (MCLLP)
- **MCRP 7-20A.4** - Evaluations and Assessments

## Installation

This is a static web application. Simply serve the files from any web server or open `index.html` directly in a browser.

For offline use, add to your home screen on mobile devices.
